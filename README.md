# Capybara Helpdesk

A complete Full-Stack management system for support tickets, made using Python, Flask and MySQL. The app allows users to report issues and the support team to solve them using a simple, clean and calm interface (just like a Capybara).

## Features
** Secure Authentification System: Sign Up, Login and Logout using crypted sessions.
** Route Protection: Users that are not logged in cannot see or add tickets.
** CRUD System for tickets:
    * Users can add new tickets (Create)
    * View of the entire ticket list (Read)
    * Resolve a ticket using a single click, changing the status from "Open" to "Resolved" (Update)
** Responsive and friendly design built with HTML5 and CSS3 (no external libraries).

## Used Technologies
** Backend: Python, Flask
** Database: MySQL (connected through 'mysql-connector-python')
** Frontend: HTML5, CSS3, Jinja2 (Template Engine)
